# War of 1949: The KMT Generals (1949党国将领风云录) 🎴⚔️

> “江山如画，一时多少豪杰。一副扑克，道尽半世兴亡。”

## 📖 项目简介

本项目是一个 **历史题材的卡牌互动图鉴与对战小游戏**。

游戏基于一份流传的“国民党将领扑克天梯榜 (1949版)”，将那个动荡年代的 **54 位关键人物** 重新梳理与呈现。从“大王”蒋介石到基层的军长，等级森严，派系林立。

我们不仅是一个静态的图鉴，更通过 **Web 交互技术**，让这些历史人物在你的指尖“活”过来。你可以翻阅档案，查看他们的派系归属与生平绰号，甚至让他们在虚拟的沙场上进行一场跨时空的对决。

无需安装任何依赖，纯 HTML/CSS/JS 实现，单机即可运行。

---

## 🎨 视觉与设计

本项目采用了 **“民国档案” (Republic Archive)** 的复古设计语言：

*   **🏛️ 沉浸式界面**：米黄色纸张背景，搭配经典的宋体与楷体排版，点缀以深红印泥色，还原历史公文的质感。
*   **🎴 拟真卡牌系统**：
    *   **背面**：CSS 纯手工绘制的“青天白日”纹章卡背。
    *   **正面**：深蓝金属质感面板 + 亮蓝流光边框 + 竖排发光名讳，融合了历史厚重感与现代游戏 UI 的精细度。
    *   **交互**：丝滑的 3D 翻转动画，支持单击翻牌、整行流水灯翻牌、一键全翻。

## 🎮 玩法说明

### 1. 阅兵场 (Gallery Mode) - *默认模式*
在这里，54 位将领按照严格的 **天梯等级 (Hierarchy)** 排列，你可以一览无余地看到当年国军的组织架构。

*   **天梯视图**：按照 `Joker > 2 > A > K ... > 3` 的顺序分层展示。点击每一行的空白处，可以体验整行卡牌像多米诺骨牌一样翻转的快感。
*   **派系视图**：点击顶部导航栏切换，可按 **【黑桃·王牌核心】**、**【红桃·军政核心】**、**【梅花·财经后继】**、**【方片·财阀统帅】** 四大派系重新洗牌展示。
*   **人物档案**：翻开任意卡牌，再次点击正面，可查看该将领的详细头衔、绰号（如“土木系领袖”、“救火队长”）以及高清肖像（支持自定义）。

### 2. 演武厅 (Battle Mode)
这是一个基于经典扑克游戏 **"War" (比大小)** 的趣味对战模式。

*   点击 **“随机对决”** 进入战场。
*   点击 **“抽取卡牌”**，系统将随机从 54 张牌中抽取两名将领进行 PK。
*   **判定规则**：严格遵循天梯等级。比如抽到“黑桃A 陈诚”对阵“方片5 黄维”，系统会判定陈诚获胜，并生成战报。
    *   *胜利！陈诚 压制了 黄维*
    *   *失败！邱清泉 不敌 李宗仁*

## 🛠️ 自定义与扩展

### 🖼️ 添加人物肖像
为了保持项目的轻量化，默认使用“☀”纹章作为人物占位。如果你想让游戏更完美，可以轻松替换为真实历史照片：

1.  准备 54 张将领的肖像图片 (推荐比例 3:4)。
2.  打开 `js/data.js` 文件。
3.  找到对应的将领数据，在 `avatar` 字段填入图片路径：
    ```javascript
    { 
        id: 's_1', 
        name: '陈诚', 
        // ...
        avatar: 'assets/images/chen_cheng.jpg' // 填入你的图片路径
    }
    ```
4.  刷新页面，代码会自动识别并以完美的“底部渐变遮罩”样式渲染图片。
