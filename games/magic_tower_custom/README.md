# 🏰 经典魔塔 - 10层试炼 (Classic Magic Tower - Web Edition)

> 一个基于 HTML5/CSS3/JavaScript 的经典策略闯关游戏复刻版。

## 📖 项目简介

本项目是一个纯前端实现的“魔塔”小游戏（Magic Tower）。游戏还原了经典的爬塔玩法，玩家扮演一名皇家骑士，进入神秘的魔塔，通过策略性的战斗、资源管理（钥匙、血量、金币），最终击败魔王救出公主。

项目完全采用原生 Web 技术构建（无第三方框架），结构清晰，无需外部图片素材（使用 CSS 与 Emoji 渲染），开箱即用。

## ✨ 主要特性

*   **10层关卡**：包含自动生成的10层地图，具备迷宫、暗道、楼梯和守卫。
*   **沉浸式开场**：复古风格的滚动字幕剧情介绍。
*   **完整数值系统**：包含生命值(HP)、攻击(ATK)、防御(DEF)、金币、经验及三色钥匙系统。
*   **战斗预测**：走到怪物身上前无需计算，系统自动演算战斗结果。
*   **怪物图鉴**：点击地图上的任意怪物，可查看其详细属性及预估战斗损伤。
*   **轻量级**：无外部图片依赖，加载速度极快，代码结构分离（HTML/CSS/JS）。

## 🎮 游戏玩法

### 1. 操作方式
*   **移动**：使用键盘 `↑` `↓` `←` `→` 方向键控制勇士移动。
*   **查看信息**：使用鼠标 **点击** 地图上的怪物图标，弹出怪物属性及战斗预测窗口。
*   **开始游戏**：在开场剧情界面点击屏幕任意位置即可进入第一层。

### 2. 核心机制
魔塔不是动作游戏，而是**数值策略游戏**。
*   **战斗**：当你触碰怪物时，若你的攻击力 > 怪物防御力，则战斗自动发生。
*   **伤害计算**：`我方受到的总伤害 = (怪物攻击 - 勇士防御) * 回合数`。
*   **资源**：
    *   🍷 **药水**：恢复生命值。
    *   💎 **宝石**：红宝石增加攻击，蓝宝石增加防御。
    *   🗝️ **钥匙**：黄色/蓝色/红色钥匙分别打开对应颜色的门。

## 🛠️ 二次开发指南

如果你想修改或扩展这个游戏，请参考以下说明：

### 1. 修改地图
打开 `script.js`，找到 `initMaps()` 函数。
`maps` 数组存储了每一层的二维数组数据。
*   `0`: 空地
*   `1`: 墙壁
*   `2`: 勇士
*   `40+`: 各种怪物 ID

### 2. 添加新怪物
在 `script.js` 中：
1.  在 `ID` 对象中定义新的怪物 ID。
2.  在 `ASSETS` 对象中设置怪物的 Emoji 图标。
3.  在 `MONSTERS` 对象中配置怪物的数值（HP, ATK, DEF, etc.）。

### 3. 替换素材
如果你想使用像素图片代替 Emoji：
在 `style.css` 中，找到对应的 `.cell` 或怪物类名，添加背景图片样式：
```css
/* 示例：替换墙壁 */
.wall {
    background-image: url('assets/wall.png');
    background-size: cover;
    border: none;
}
```
