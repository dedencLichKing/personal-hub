### 设计说明：

1.  **剧情与背景**：

    *   游戏启动页介绍了“江西女友被困缅北3K园区”的背景。

    *   **Level 1 (丛林)**: 背景深绿色，地面主要是土路。

    *   **Level 2 (园区)**: 背景深蓝色，模拟高楼和建筑，地形会有较大的高低落差（模拟楼层）。

    *   **Level 3 (仓库)**: 背景暗红色，模拟内部环境，最终会有Boss战。

2.  **操作与手感**：

    *   **移动**: `WASD`。

    *   **跳跃**: 区分了 `I` (小跳，用于精密跳跃) 和 `K` (大跳，用于跨越深坑)。

    *   **射击**: `U` 是单点（手速快可以点得快），`J` 是按住连发（自动步枪模式）。

    *   **不可回头**: 摄像机 `camera.x` 只增不减，模拟经典街机卷轴体验。

3.  **敌人系统 (AI)**：

    *   **杂兵 (Type 0-1)**: 只会傻傻地向左跑（冲锋）。

    *   **射击兵 (Type 2-4)**: 看到玩家会停下并射击。

    *   **炮台 (Type 5-7)**: 不动，血量稍高，会发射瞄准玩家的子弹。

    *   **机甲 (Type 8-9)**: 血厚，发射大子弹。

    *   **Boss (Type 10)**: 最终关出现，有特定移动规律和弹幕。

4.  **武器系统**：

    *   参考了《魂斗罗》。

    *   **S (Spread)**: 散弹，一次发射三发，扇形攻击。

    *   **M (Machine)**: 射速极快，无冷却。

    *   **L (Laser)**: 激光，速度快，可穿透敌人，伤害高。

    *   杀死敌人有概率掉落带有字母的方块，拾取即可切换武器。

5.  **难度系统**：

    *   在开始界面按 1/2/3 选择。

    *   影响敌人的血量倍率和得分倍率。

6.  **技术实现**：

    *   使用 HTML5 Canvas API 进行绘制。

    *   没有使用任何外部图片，所有图形都是代码绘制的（矩形、文字），保证了代码的便携性，复制粘贴即可运行。

    *   包含简单的 AABB 碰撞检测和物理重力模拟。
